Appendix E
Cartool Options

 

These options are called from the menu File | Options .

 

Windows registry

    Reset to default

      Rewrite brand new file associations in the registry (which tells Windows how to open files from the Explorer). Very useful when old versions, deleted versions, or multiple versions of Cartool live together and compete for the privilege of opening your files! Double-clicking on a file doesn't work anymore? so this is time to reset the file associations but from the correct version of Cartool only .

    Delete all

      This will remove from the registry all the file associations done by Cartool. Be careful, if you are quitting Cartool then running it again, the registry will be again filled with the file associations as it is (bad?) its habit.
