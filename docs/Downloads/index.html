Templates MRIs:

MNI152.NlinAsym20

The  MNI152.NlinAsym20.Kit.zip  is a "starter pack" which contains all you need to start working with the MNI template in Cartool (updated 2024-07-08):

  • The latest MNI Asymmetric Template Extended Head & Brain 2020

  • A Grey Mask, generated by Cartool, suited for inverse solutions computation

  • A Tissues Atlas of 14 different segmented tissues (grey, white, skull, CSF, blood, eye, air etc...)

  • A set of Solution Points, with a handy 6mm spacing

  • A license text file

Note: ROIs are not currently available for this model, as the template brain differs slightly from the 2009 version.

MNI152.NlinAsym09c

The  MNI152.NlinAsym09c.Kit.zip  is a "starter pack" which contains all you need to start working with the MNI template in Cartool (updated 2024-07-08):

  • The original MNI Asymmetric Template Head & Brain 2009c

  • A Grey Mask, generated by Cartool, suited for inverse solutions computation

  • A Tissues Atlas of 14 different segmented tissues (grey, white, skull, CSF, blood, eye, air etc...)

  • The latest AAL Atlas (AAL3v1), with the origin properly set

  • A set of Solution Points, with a handy 6mm spacing

  • ROIs file generated from the AAL and the solution points above

  • A license text

AAL Atlases

The  AAL Atlases  zip file contains:

  • All versions of the AAL atlas (1, 2, 3 and 3v1), with the origin properly set (the latest AAL is already included in the kit above)
  • A reference text file
